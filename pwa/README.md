# HAnWord PWA

这是手机可安装版本，和仓库根目录下的桌面 Python 版分开维护。

## 特点

- 在浏览器本地计算，不上传汉字输入
- 可添加到手机主屏幕，离线使用
- 支持 `设备`、`应用`、`网站`、`其他用途` 四种模式
- 使用与桌面版一致的 `PBKDF2-HMAC-SHA256` 派生流程
- 内置安装提示、版本号显示和离线缓存更新机制

## 与桌面版的关系

- 根目录版本：桌面/命令行 Python 版，使用 `PBKDF2-HMAC-SHA256`
- `pwa/` 版本：移动端离线 PWA，使用同一套 `PBKDF2-HMAC-SHA256` 配置
- 现在两个版本在相同输入下应生成**完全一致**的结果

## 本地预览

在 `pwa/` 目录启动一个静态文件服务，例如：

```bash
cd pwa
python3 -m http.server 8080
```

然后在浏览器打开 `http://localhost:8080`。

如果要在手机上测试：

1. 让手机和电脑在同一局域网
2. 用电脑 IP 访问 `http://<你的电脑IP>:8080`
3. 在浏览器菜单中选择“添加到主屏幕”

## 更新与缓存

- 页面顶部会显示当前 PWA 版本号
- 检测到新版本资源时，会出现“更新到最新版”按钮
- 如果浏览器缓存异常，可使用“清理离线缓存”后重新加载
- 升级 `app.js`、`sw.js`、`manifest.webmanifest` 时，记得同步递增版本号
